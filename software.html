Software

Current Software includes Dustin's "Get Fit With League of Legends" Program.

 

This software retrieves an users information from Riot Games API and returns the recent scores from his/her's match history, as well as their teammates and calculates the difference between the scores. This has a multiplier based on the difficulty of workout they would prefer as well as a selector for the region they play in. Updates and additions will be coming soon for this program.
